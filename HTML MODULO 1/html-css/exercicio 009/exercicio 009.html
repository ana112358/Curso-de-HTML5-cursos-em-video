Citações e códigos

Citações

Pa uma citação simple use 'q' como Segundo Ana Paula A vida é linda

Para citações completas utilize "blockquote cite='dagdfhdh'", como

Segundo um filósofo

A vida é bela

Código

Utilizamos o 'code' para as letra terem o mesmo espaço: ShiTeiiiL(sem code) ShiTeiiiL(com code)

Para aparecer a identação é so usar o "pre" como : 

 
    Este é 
    um texto 
    com 
    identação.
